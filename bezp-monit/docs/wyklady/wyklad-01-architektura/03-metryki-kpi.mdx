# Metryki i KPI

## Kluczowe wskaźniki wydajności

### Performance Ratio (PR)
- **Definicja**: Stosunek rzeczywistej energii do energii teoretycznej
- **Wzór**: PR = E_rzeczywista / E_teoretyczna × 100%
- **Cel**: > 85% dla instalacji PV

### Capacity Factor (CF)
- **Definicja**: Stosunek średniej mocy do mocy nominalnej
- **Wzór**: CF = P_średnia / P_nominalna × 100%
- **Typowe wartości**: 15-25% (PV), 25-45% (wiatr)

## Metryki techniczne

### Availability
- **Definicja**: Czas pracy / Czas całkowity × 100%
- **Cel**: > 99% dla systemów krytycznych

### MTBF/MTTR
- **MTBF**: Mean Time Between Failures
- **MTTR**: Mean Time To Repair
- **Cel**: MTBF > 720h, MTTR < 4h

## Monitoring jakości danych

### Data Quality Index (DQI)
- **Completeness**: % brakujących danych
- **Accuracy**: Dokładność pomiarów
- **Consistency**: Spójność czasowa
