# Przepływ, poziom, ciśnienie

## Przepływ

- Metody: elektromagnetyczne, ultradźwiękowe, Coriolisa, DP (tensometr + zwężka).
- Kryteria doboru: lepkość medium, zakres przepływów, czystość, strata ciśnienia.

## Poziom

- Hydrostatyczne, radarowe (FMCW), ultradźwiękowe, pływakowe; kompensacja temperatury.

## Ciśnienie

- Przetworniki piezorezystancyjne/pojemnościowe; membrany separacyjne dla mediów korozyjnych.

## Przykład

Zbiornik fermentera: czujnik poziomu radarowy + czujnik ciśnienia gazu zapewniły stabilność procesu; alarm różnicy ciśnień ostrzegł o niedrożności odmy.
